## 第二遍：风格审校（AI味降重）⭐⭐⭐

### 1. 删除套话
- [ ] 删除"在当今时代"
- [ ] 删除"综上所述"
- [ ] 删除"值得注意的是"
- [ ] 删除"需要强调的是"
- [ ] 删除"总而言之"
- [ ] 删除其他AI套话

**修改位置**：
```
段落X：删除了"XXX"
段落Y：删除了"XXX"
```

### 2. 拆解AI句式
- [ ] 检查"不是…而是…"是否过多
- [ ] 检查"不仅…还…"是否重复
- [ ] 检查三段论是否过于工整
- [ ] 拆分过于对称的句式

**修改位置**：
```
段落X：拆解了"不是A而是B"句式
段落Y：调整了过于对称的结构
```

### 3. 替换书面词汇
- [ ] "显著提升" → 具体数字/描述
- [ ] "充分利用" → "用好"
- [ ] "进行操作" → 具体动词
- [ ] "相关工作" → 具体说做什么
- [ ] "有效提升" → 具体效果
- [ ] 其他书面语 → 口语化

**修改示例**：
```
❌ 原文："显著提升了效率"
✅ 改为："每个项目省3-5小时"

❌ 原文："充分利用AI能力"
✅ 改为："用好AI"
```

### 4. 改成口语化
- [ ] "通过XX方式进行XX" → 直接说怎么做
- [ ] 过于正式的表达 → 像说话一样写
- [ ] 检查是否像本人在说话

**修改位置**：
```
段落X：将"通过...进行..."改为直接表达
段落Y：简化了正式表达
```

### 5. 加入真实细节
- [ ] 抽象表达 → 具体数字
- [ ] 抽象表达 → 具体案例
- [ ] 泛泛而谈 → 真实场景
- [ ] "效率提升" → 具体量化

**修改示例**：
```
❌ 原文："大大提升了生产力"
✅ 改为："原来要2天的工作，现在半天就能搞定"
```

### 6. 加入个人态度
- [ ] 中立客观 → 明确观点
- [ ] "可能会" → "我觉得"
- [ ] 增加个人判断
- [ ] 表达真实感受

**修改位置**：
```
段落X：加入了个人观点"我觉得..."
段落Y：表达了真实感受
```
