## 二、什么是Claude Skills？用人话讲

Claude Skills不是插件，也不是扩展。

它更像是"给Claude报了个培训班"。

你知道Claude本身很聪明，能写代码、能分析数据、能写文章。但它不会处理PDF，不会生成PPT，不会创建Excel公式……这些具体的"技能"，它原生是不具备的。

Skills就是教Claude学会这些新技能的方式。

Anthropic官方开源了16个Skills，涵盖文档处理、设计创意、开发技术等场景。GitHub上51.4k stars，4.9k forks，活跃度相当高。

更重要的是，这些Skills里有4个是Claude生产环境实际在用的：**pdf、docx、xlsx、pptx**。也就是说，这4个是官方亲自验证过、推荐给你用的。

今天这篇文章，我就挑最实用的5个Skills，手把把手教你怎么用它们提升办公效率。

不讲技术细节，只讲"能帮你做什么"和"怎么用"。

## 三、必备技能1：PDF处理（pdf skill）⭐⭐⭐

### 能做什么

PDF是办公场景里最常见的文件格式。合同、发票、报告、论文……全是PDF。

但PDF最烦的地方就是：难编辑、难提取。

你想把PDF里的表格复制到Excel？手动一个个复制，格式全乱了。你想提取PDF里的文字？复制出来全是换行符和乱码。

pdf skill就是专门解决这个痛点的。

它能做3件事：
1. **提取PDF内容**：把PDF里的文字、表格提取出来，格式整理好
2. **填写PDF表单**：自动填写那些需要手填的PDF表格
3. **格式转换**：把PDF转成Markdown、Word等格式

### 真实使用场景

**场景1：提取合同关键信息**

你收到一份20页的合同PDF，需要提取甲方、乙方、签约日期、金额这些关键信息。

传统做法：一页页翻，手动复制粘贴，容易漏掉。

用pdf skill：一条指令搞定。
```
使用pdf skill提取这份合同的甲方、乙方、签约日期、合同金额、付款方式，整理成表格。
```

Claude会自动读取PDF，提取关键信息，给你一个结构化的表格。

**场景2：批量处理发票**

财务每个月要处理几十张发票PDF。手动提取发票号、金额、税额，累死人。

用pdf skill：批量提取，直接生成Excel。

### 适用人群

- 财务（处理发票、合同）
- 法务（分析合同条款）
- 研究人员（提取论文数据）
- 所有需要处理PDF的人

## 四、必备技能2：Word文档（docx skill）⭐⭐⭐

### 能做什么

Word文档是办公三件套之一。写周报、写报告、写合同，全靠它。

但写Word最烦的是什么？格式调整、样式统一、批量修改……这些重复性工作占了大量时间。

docx skill能做3件事：
1. **创建Word文档**：自动生成格式规范的文档
2. **批量修改**：统一调整字体、段落、标题格式
3. **内容填充**：根据模板自动生成周报、月报

### 真实使用场景

**场景1：自动生成周报**

每周五下午写周报，最烦的是什么？不是没内容，是格式麻烦。

标题用几号字？段落间距多少？每次都得调。

用docx skill：
```
使用docx skill创建一个周报文档：
标题：第3周工作周报
本周完成事项：
1. 完成Q1销售数据分析
2. 修复客户反馈的3个bug
3. 参加产品评审会议
下周计划：
1. 启动Q2项目规划
2. 优化系统性能
格式：标题用一号字加粗，正文用五号字，行距1.5倍
```

Claude自动生成格式规范的Word文档，你只需要填内容。

**场景2：批量修改格式**

你写了一份50页的报告，老板说"把所有一级标题改成黑体三号字"。

传统做法：一个个找，一个个改。

用docx skill：一条指令搞定。

### 适用人群

- 所有需要写Word文档的人（尤其是经常写周报、月报的）

## 五、必备技能3：Excel操作（xlsx skill）⭐⭐⭐

### 能做什么

Excel是数据分析的核心工具。但很多人用Excel，只会简单的加减乘除。

透视表？公式嵌套？图表美化？这些高级功能，学起来门槛太高。

xlsx skill能做3件事：
1. **数据分析**：自动计算求和、平均值、增长率
2. **图表生成**：根据数据自动生成柱状图、折线图、饼图
3. **公式计算**：帮你写复杂的Excel公式

### 真实使用场景

**场景1：销售数据分析**

你有一份销售数据表，包含日期、产品、销售额。老板让你分析：
- 哪个产品卖得最好？
- 销售趋势如何？
- 生成一个图表展示

传统做法：手动做透视表，手动插入图表，调格式调半天。

用xlsx skill：
```
使用xlsx skill完成以下任务：
1. 计算每个产品的总销售额和平均销售额
2. 找出销售额最高的3个产品
3. 生成一个柱状图，展示每个产品的总销售额对比
4. 生成一个折线图，展示近30天的销售趋势
```

Claude自动完成所有计算和图表生成，你只需要看结果。

**场景2：财务报表**

每个月做财务报表，公式一堆，容易算错。

用xlsx skill：描述你要算什么，Claude帮你写公式、填数据、生成报表。

### 适用人群

- 销售（数据分析）
- 财务（报表生成）
- 运营（数据监控）
- 所有需要分析数据的人

第3页：重点项目 - 项目A：进度90%，项目B：已上线

Claude自动生成5页PPT，格式统一，你只需要微调。

**场景2：培训材料**

## 九、如何安装和使用Skills？（保姆级教程）

前面讲了这么多，你可能会问："听起来很厉害，但我怎么用？"

别担心，我一步步教你。

### 第一步：确认Claude Code版本

Skills功能是Claude Code 2.1及以上版本才有的。

打开终端，输入：
```
claude --version
```

如果显示2.1.0或更高，说明你可以用了。

如果版本低于2.1，升级一下：
```
npm update -g @anthropic-ai/claude-code
```

### 第二步：下载Skills

官方Skills仓库：https://github.com/anthropics/skills

你可以：
- 方式1：直接下载整个仓库，解压到 `~/.claude/skills/` 目录
- 方式2：只下载你需要的Skills（比如pdf、docx、xlsx、pptx），复制到 `~/.claude/skills/` 目录

### 第三步：调用Skills

在Claude Code中，直接说：
```
使用pdf skill提取这个PDF文件的内容
```

或者：
```
使用xlsx skill分析这个Excel文件，生成图表
```

Claude会自动调用对应的Skill完成任务。

### 常见问题

**Q1：我不会编程，能用Skills吗？**

能。这5个Skills（pdf、docx、xlsx、pptx、frontend-design）都不需要编程背景。你只需要用自然语言描述你要做什么，Claude会帮你完成。

**Q2：Skills收费吗？**

官方的这16个Skills全部免费开源。

但你需要有Claude Code的使用权限（Claude API或Claude Pro订阅）。

**Q3：Skills和MCP有什么区别？**

简单说：
- Skills：教Claude学会新技能（比如处理PDF）
- MCP：让Claude连接外部工具（比如连接数据库、调用API）

Skills更轻量，MCP更灵活。办公场景用Skills就够了。

**Q4：如果Skill出错怎么办？**

检查两件事：
1. Skill文件是否正确放在 `~/.claude/skills/` 目录下
2. Claude Code版本是否是2.1及以上

如果还是不行，到GitHub仓库提Issue，官方响应很快。

## 十、为什么现在是用Skills的最佳时机？

Claude Code 2.1刚发布（2026年1月7日），带来了Skills的三大升级：

### 1. 热重载：调试效率翻倍

以前你改了Skill文件，必须重启Claude Code才能生效。调试一个Skill可能要重启十几次，体验很差。

现在改完Skill，立刻生效，不用重启。

这对想自定义Skills的人来说是个大提升。你可以快速迭代，快速测试，大大降低了试错成本。

### 2. Context隔离：Skill之间不干扰

以前多个Skills同时运行，可能会互相影响。

现在每个Skill可以在独立的上下文中运行（`context: fork`），不会污染主对话。

### 3. Hooks支持：更灵活的控制

现在Skills支持Hooks（PreToolUse、PostToolUse、Stop），可以在工具调用前后插入自定义逻辑。

这让Skills的玩法更灵活了。

这三个功能，让Skills从"能用"变成"好用"。
