## ✅ 素材使用原则

### 必须遵守
1. ✅ **真实性**：所有素材都必须是真实的
2. ✅ **改写**：不能直接复制粘贴，要改写成长文逻辑
3. ✅ **自然融入**：要自然融入文章，不能生硬插入
4. ✅ **标注来源**：内部记录素材来源（便于追溯）

### 禁止操作
1. ❌ **编造**：不能编造数据或夸大事实
2. ❌ **直接复制**：不能直接把短文复制成长文
3. ❌ **无关素材**：不能硬塞无关的素材

## 📝 素材整理指南

### X平台内容整理

**步骤**：
1. 导出自己的X推文（手动或使用工具）
2. 筛选出有价值的内容：
   - ✅ 真实观点
   - ✅ 真实体验
   - ✅ 有趣的洞察
   - ❌ 转发内容（除非有自己的评论）
3. 按主题分类整理
4. 更新索引文件

**格式示例**：
```markdown
## 关于Claude Code的观点

**发布日期**：2024-XX-XX  
**原文链接**：https://x.com/xxx  

**内容**：
[原始内容]

**可用场景**：
- 文章开头引入
- 观点支撑

**关键词**：Claude Code, AI编程, 效率
```

### 小红书内容整理

**步骤**：
1. 整理自己的小红书笔记
2. 筛选出有价值的内容
3. 按主题分类
4. 更新索引

**格式**：同X平台

### 公众号历史文章整理

**步骤**：
1. 筛选出自己满意的历史文章
2. 标注每篇文章的特点：
   - 写作风格
   - 核心观点
   - 可复用的案例
3. 提取可复用的素材
